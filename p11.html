CSS Pseudo-elements

hw css pseudo elements allow you to style the elements or parts of the elements without using any ids or classes to them

css3 introduced a new double-colon(::) syntax for pseudo elements to distinguish between them and pseudo-classes

selector::pseudo-elements {property value;}
